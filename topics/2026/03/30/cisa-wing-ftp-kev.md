# CISAがWing FTPサーバーの脆弱性CVE-2025-47813をKEVカタログに追加、即時パッチ適用を要求
Tags: Security

- CISA Flags Actively Exploited Wing FTP Vulnerability Leaking Server Paths (2026-03-28)
  https://thehackernews.com/2026/03/cisa-flags-actively-exploited-wing-ftp.html

CISAはWing FTPサーバーのサーバーパス漏洩脆弱性CVE-2025-47813を既知悪用脆弱性（KEV）カタログに追加し、連邦政府機関に対して3月30日までのパッチ適用を義務付けた。この欠陥を悪用することでサーバーのファイルシステム構造が露出し、さらなる攻撃の足がかりとなる危険性がある。
