# Kubernetes移行ツール「Ingress2Gateway 1.0」正式リリース、Ingress-NGINX廃止に備える
Tags: Cloud, OSS

- Ingress2Gateway 1.0 Release (2026-03-23)
  https://kubernetes.io/blog/2026/03/20/ingress2gateway-1-0-release/

Ingress-NGINXの廃止に伴う移行を支援するツール「Ingress2Gateway 1.0」が正式リリースされた。30以上のIngress-NGINXアノテーションに対応し、既存のIngressリソースをKubernetes Gateway APIへ自動変換する機能を提供する。Kubernetes 1.36（2026年4月22日リリース予定）ではGateway APIの本格採用が進む予定。
