# Amazon Route 53 Global ResolverがGA、DNS over HTTPS/TLSで30リージョンに対応
Tags: Cloud

- Amazon Route 53 Global Resolver is now generally available (2026-03-23)
  https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-route-53-global-resolver/

AWSがAmazon Route 53 Global Resolverを30リージョンで一般提供開始した。オンプレミスやリモートクライアントを含む任意の場所から、DNS-over-HTTPS/TLSによる暗号化通信でプライベートおよびパブリックドメインの解決が可能になり、ハイブリッドクラウド環境のDNSセキュリティと信頼性が大幅に向上した。
