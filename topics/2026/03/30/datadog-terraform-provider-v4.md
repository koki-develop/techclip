# Datadog Terraform Provider v4.0.0リリース、AWS統合リソース統合とモニター制御を刷新
Tags: Cloud

- Datadog Terraform Provider v4.0.0 (2026-03-23)
  https://www.datadoghq.com/blog/datadog-terraform-provider-v4/

DatadogはTerraformプロバイダーのメジャーバージョンv4.0.0をリリースした。予測可能なモニターアクセス制御、4つのレガシーリソースを統合した新しいAWS統合リソース、セキュリティ標準の強化など、運用管理の大幅な改善を提供する。
