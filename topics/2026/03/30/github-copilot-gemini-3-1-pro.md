# GitHub CopilotがGemini 3.1 ProをJetBrains・Xcode・Eclipseに拡大、旧Gemini 3 Proを非推奨化
Tags: OSS, AI

- Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse (2026-03-23)
  https://github.blog/changelog/2026-03-23-gemini-3-1-pro-is-now-available-in-jetbrains-ides-xcode-and-eclipse/
- Gemini 3 Pro deprecated (2026-03-26)
  https://github.blog/changelog/2026-03-26-gemini-3-pro-deprecated/

GitHub CopilotのGemini 3.1 ProがJetBrains IDE、Xcode、Eclipseでも利用可能となり、対応IDEが大幅に拡大した。合わせて旧バージョンのGemini 3 Proが非推奨化され、Copilotユーザーはより高性能な3.1 Proへの移行が促される。
