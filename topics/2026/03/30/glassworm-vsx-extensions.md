# GlassWormが72件のOpen VSX拡張機能を悪用、SolanaをC2に使う開発者標的攻撃
Tags: Security, OSS

- GlassWorm Supply-Chain Attack Abuses 72 Open VSX Extensions to Target Developers (2026-03-28)
  https://thehackernews.com/2026/03/glassworm-supply-chain-attack-abuses-72.html
- GlassWorm malware hits 400+ code repos on GitHub, npm, VSCode, OpenVSX (2026-03-28)
  https://www.bleepingcomputer.com/news/security/glassworm-malware-hits-400-plus-code-repos-on-github-npm-vscode-openvsx/

GlassWormと名付けられたサプライチェーン攻撃キャンペーンが72件のOpen VSX拡張機能を悪用し、GitHub・npm・VSCode・OpenVSXの400以上のリポジトリに及ぶことが明らかになった。Solanaブロックチェーンをコマンド＆コントロールインフラとして利用する斬新な手法で、開発者環境から認証情報・暗号資産ウォレット・環境シークレットを窃取する機能を持つ。
