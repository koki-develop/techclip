# TypeScript 6.0がリリース、JavaScriptベース最後のバージョンとしてGo実装へ移行準備
Tags: Programming Languages

- TypeScript 6.0 Ships as Final JavaScript-Based Release, Clears Path for Go-Native 7.0 (2026-03-23)
  https://visualstudiomagazine.com/articles/2026/03/23/typescript-6-0-ships-as-final-javascript-based-release-clears-path-for-go-native-7-0.aspx

MicrosoftがTypeScript 6.0をリリースし、これがJavaScriptベースの最後のバージョンとなった。今後はGoで実装されたTypeScript 7.0へ移行するための非推奨化と移行フラグが導入されており、TypeScriptのランタイム刷新に向けた大きな転換点となる。
