# オランダ国家警察がフィッシング攻撃によるセキュリティ侵害を公表
Tags: Security

- Dutch Police discloses security breach after phishing attack (2026-03-27)
  https://www.bleepingcomputer.com/news/security/dutch-police-discloses-security-breach-after-phishing-attack/

オランダ国家警察（Politie）は、フィッシング攻撃を受けてセキュリティ侵害が発生したことを公表した。被害の影響は限定的で市民データへの影響はないとしているが、攻撃者のアクセスはすでに遮断済みで調査が進められている。
