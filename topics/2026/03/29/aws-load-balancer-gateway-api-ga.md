# AWS Load Balancer ControllerがKubernetes Gateway APIのGA（一般提供）サポートをリリース
Tags: Cloud

- AWS Load Balancer Controller Reaches GA with Kubernetes Gateway API Support (2026-03-25)
  https://www.infoq.com/news/2026/03/aws-gateway-api-ga/
- AWS Load Balancer Controller adds general availability support for Kubernetes Gateway API (2026-03-25)
  https://aws.amazon.com/blogs/networking-and-content-delivery/aws-load-balancer-controller-adds-general-availability-support-for-kubernetes-gateway-api/

AWSはKubernetes Load Balancer ControllerでGateway APIのGA（一般提供）サポートをリリースした。アノテーションベースの設定を型安全なCRDに置き換え、ALBとNLBの両方をGateway API仕様で管理できるようになり、Kubernetesネットワーキングの標準化が一歩進んだ。
