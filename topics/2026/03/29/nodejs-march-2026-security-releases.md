# Node.jsが複数バージョンにセキュリティリリース、9件のCVEを修正
Tags: Security, Programming Languages

- Node.js — Tuesday, March 24, 2026 Security Releases (2026-03-24)
  https://nodejs.org/en/blog/vulnerability/march-2026-security-releases

Node.jsの20.x、22.x、24.x、25.x全アクティブラインに対してセキュリティリリースが行われ、TLS SNIコールバックのクラッシュやHTTP/2メモリリーク、V8のHashDoSなど高・中・低の計9件のCVEが修正された。
