# GitHubが2026年OSSレポートを公開、AIによる低品質コントリビューション増加がメンテナの新課題に

Tags: OSS

- GitHub's Points to a More Global, AI-Challenged Open Source Ecosystem in 2026 (2026-03-24)
  https://www.infoq.com/news/2026/03/github-ai-2026/

GitHubが2026年のオープンソースエコシステムに関するレポートを公開し、2025年に約3,600万人の新規開発者が参加した一方で、AIが生成した低品質なコントリビューション（"AI slop"）の増加がプロジェクト維持管理における新たな課題となっていると報告した。世界的な開発者コミュニティの拡大とAI活用の普及が同時進行する中、OSSエコシステムの持続可能性が問われる状況だ。
