# Rust 1.94.1リリース、1.94.0の3つのリグレッションとCargoセキュリティCVEを修正

Tags: Programming Languages

- Announcing Rust 1.94.1 (2026-03-26)
  https://blog.rust-lang.org/2026/03/26/1.94.1-release/

Rust 1.94.0で発生したWebAssemblyスレッド対応・Windows API互換性・Clippyの内部エラーという3つのリグレッションを修正したメンテナンスリリース。加えてCargoが使用するtarクレートのセキュリティCVEへの対応も含まれており、1.94.0ユーザーへのアップデートが推奨されている。
