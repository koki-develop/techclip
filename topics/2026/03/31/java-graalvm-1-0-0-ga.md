# GraalVM Native Build Tools 1.0.0がGA、Javaエコシステムで複数の主要リリースが相次ぐ

Tags: Programming Languages

- Java News Roundup: GraalVM Build Tools, EclipseLink, Spring Milestones, Open Liberty, Quarkus (2026-03-30)
  https://www.infoq.com/news/2026/03/java-news-roundup-mar23-2026/

GraalVM Native Build Tools 1.0.0のGA版リリースやEclipseLink 5.0.0（Jakarta EE 11対応）など、Javaエコシステムの複数のフレームワークが同週にリリースされた。Spring Boot 4.1.0の第4マイルストーンやQuarkusのポイントリリースも含まれており、ネイティブコンパイルとクラウドネイティブ対応を中心としたJavaプラットフォームの積極的な進化が続いている。
