# VercelがAI駆動UI生成フレームワーク「json-render」をオープンソースで公開

Tags: Programming Languages, OSS

- Vercel Releases JSON-Render: a Generative UI Framework for AI-Driven Interface Composition (2026-03-26)
  https://www.infoq.com/news/2026/03/vercel-json-render/

VercelがオープンソースのGenerative UIフレームワーク「json-render」を公開した。AIモデルがZodスキーマで定義されたコンポーネントカタログに基づいてJSON仕様を生成し、React・Vue・Svelte・Solid・React Nativeなど複数のフロントエンドフレームワーク向けに動的なUIをレンダリングできる仕組みで、AIエージェントによるUI自動生成の新たなアプローチとして注目されている。
