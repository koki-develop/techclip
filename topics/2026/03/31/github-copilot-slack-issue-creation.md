# GitHub CopilotがSlack連携でIssue作成に対応、自然言語でリポジトリに直接登録可能に
Tags: OSS, AI

- Create issues from Slack with Copilot (2026-03-30)
  https://github.blog/changelog/2026-03-30-create-issues-from-slack-with-copilot/

GitHub CopilotがSlackアプリと連携し、自然言語でGitHub Issueを直接作成できる新機能が追加された。SlackチャンネルでGitHubアプリに@メンションするだけで、タイトル・本文・担当者・ラベルなどを含む構造化されたIssueをリポジトリに登録でき、チャットから開発タスク管理へのシームレスな移行を実現する。
