# GitHubが新しいプルリクエストダッシュボードをパブリックプレビューで公開、PRインボックスと保存済みビュー機能を追加
Tags: OSS

- New pull requests dashboard is in public preview (2026-03-26)
  https://github.blog/changelog/2026-03-26-new-pull-requests-dashboard-is-in-public-preview/

GitHubがgithub.com/pullsにて新しいプルリクエストダッシュボードのパブリックプレビューを公開した。新たなPRインボックスと保存済みビュー機能により、開発者が自分に関係するPRを整理・優先順位付けして効率的に管理できるようになる。
