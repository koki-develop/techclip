# GitHub Enterprise Server 3.20 GA、Java 26対応CodeQLや自動マルウェア検出も同時リリース
Categories: OSS

- GitHub Enterprise Server 3.20 release and changelog (2026-03-27)
  https://github.blog/changelog/

GitHub Enterprise Server 3.20が一般提供を開始した。同時に、Dependabotによるnpmパッケージのマルウェア自動検出、Java 26をサポートするCodeQL 2.24.3、Actions Runner Controller 0.14.0もリリースされ、開発者のセキュリティと自動化環境が大きく強化された。
