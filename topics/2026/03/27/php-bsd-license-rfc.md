# PHPがBSD 3条項ライセンスへの移行を投票中、長年の二重ライセンス構造を解消へ
Categories: OSS, Programming Languages

- PHP Licensing RFC: Moving to BSD 3-Clause (2026-03-26)
  https://lwn.net/

Ben Ramseyが主導するPHPのライセンス簡素化RFCの投票が2026年4月4日まで行われている。現行のPHP License v3.01とZend License v2.0の二重ライセンス構造をBSD 3条項ライセンスに統一する提案であり、採択されれば他のOSSプロジェクトとの親和性が大幅に向上する。
