# GitHub CopilotのメトリクスがCoding Agentのアクティブユーザーを識別・追跡可能に
Tags: OSS

- Copilot usage metrics now identify active Copilot coding agent users (2026-03-25)
  https://github.blog/changelog/2026-03-25-copilot-usage-metrics-now-identify-active-copilot-coding-agent-users/

GitHub Copilotの利用状況メトリクスが更新され、エンタープライズおよび組織管理者がCopilot Coding Agent（CCA）のアクティブユーザーを日次・28日間レポートで把握できるようになった。IDEのエージェントモードとCopilot Coding Agentの使用を区別して追跡できる機能が追加された。
