# TeamPCPがTelnyxのPyPIパッケージを汚染、WAVファイルに認証情報窃取機能を隠蔽
Tags: Security

- TeamPCP Pushes Malicious Telnyx Versions to PyPI, Hides Stealer in WAV Files (2026-03-27)
  https://thehackernews.com/2026/03/teampcp-pushes-malicious-telnyx.html
- Telnyx PyPI Package With 742,000 downloads Compromised in TeamPCP Supply Chain Attack (2026-03-27)
  https://cybersecuritynews.com/telnyx-pypi-package-compromised/

TeamPCPと呼ばれる攻撃者グループがPyPIにtelnyxパッケージの悪意あるバージョン（4.87.1および4.87.2）を公開し、.WAVファイル内に認証情報窃取機能を隠蔽していた。累計74万2千ダウンロードを持つ人気パッケージが標的となり、影響を受けたユーザーはバージョン4.87.0へのダウングレードが推奨されている。
