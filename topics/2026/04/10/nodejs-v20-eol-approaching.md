# Node.js v20が2026年4月30日にEnd-of-Life、v24への移行が急務に
Tags: Programming Languages

- Node.js v20 is Reaching End of Life (2026-04-07)
  https://www.herodevs.com/blog-posts/node-js-v20-is-reaching-end-of-life

Node.js v20が2026年4月30日にEnd-of-Lifeを迎える。EOL後はセキュリティパッチやバグ修正が提供されなくなり、AWS LambdaなどのクラウドランタイムもNode.js 20の非推奨化を予定している。Node.js v24への移行が推奨されている。
