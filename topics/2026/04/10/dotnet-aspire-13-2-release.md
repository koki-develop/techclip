# .NET Aspire 13.2リリース、TypeScript AppHostプレビューとデタッチモードを追加
Tags: Programming Languages

- Aspire 13.2 Released with Expanded CLI, TypeScript AppHost Preview, and Dashboard Improvements (2026-04-09)

.NET Aspire 13.2がリリースされ、バックグラウンド実行のデタッチモードや複数インスタンスの並行起動に対応した。C#に加えTypeScriptでアプリケーション構成を定義できる「TypeScript AppHost」プレビューとBunサポートも追加された。
