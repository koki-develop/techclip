# Module Federation 2.0が安定版リリース、Webpack以外にRspack・Vite・Rolldownもサポート
Tags: Programming Languages, OSS

- Module Federation 2.0 Reaches Stable Release with Wider Support outside of Webpack (2026-04-03)
  https://www.infoq.com/news/2026/04/module-federation-2-stable/
- MF 2.0 Stable Release: Balancing Developer Productivity and Extreme Performance (2026-04-03)
  https://module-federation.io/blog/v2-stable-version

Module Federation 2.0が安定版としてリリースされ、Webpack以外にRspack・Vite・Rolldownなど多様なバンドラーをサポートした。動的TypeScript型ヒントの自動生成やNode.jsサーバーサイドレンダリング対応など、マイクロフロントエンド開発を大幅に強化する機能が追加された。
