# Djangoセキュリティリリース公開、5件の脆弱性を修正しバージョン4.2は延長サポート終了
Tags: Security, Programming Languages

- Django security releases issued: 6.0.4, 5.2.13, and 4.2.30 (2026-04-07)
  https://www.djangoproject.com/weblog/2026/apr/07/security-releases/

Djangoのセキュリティリリース（6.0.4、5.2.13、4.2.30）が公開された。ASGIリクエストのヘッダースプーフィング（CVE-2026-3902）、GenericInlineModelAdminの権限バイパス（CVE-2026-4277）、MultiPartParserのDoS脆弱性（CVE-2026-33033）など計5件のセキュリティ問題が修正された。Django 4.2はこのリリースをもって延長サポートが終了となる。
