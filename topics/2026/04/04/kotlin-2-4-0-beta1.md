# Kotlin 2.4.0-Beta1リリース、コンテキストパラメーターの安定化とSwift Packageサポートを追加
Tags: Programming Languages

- What's new in Kotlin 2.4.0-Beta1 (2026-03-31)
  https://kotlinlang.org/docs/whatsnew-eap.html

Kotlin 2.4.0-Beta1がリリースされ、コンテキストパラメーターの安定化、`UInt`/`ULong`から`BigInteger`への変換API、Swift PackageをKotlin/Nativeの依存関係として利用可能にする機能など多数の改善が含まれる。Kotlin/Native・JS・WasmでのKlib内部インライン化もデフォルト有効化され、マルチプラットフォーム開発がさらに強化された。
