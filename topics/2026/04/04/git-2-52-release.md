# Git 2.52リリース、RustコードをGitに初統合しGit 3.0への準備が本格化
Tags: OSS

- Git 2.52 Released With More Preparations Toward Git 3.0 (2026-04-03)
  https://www.phoronix.com/news/Git-2.52-Released

Git 2.52がリリースされ、SHA-256相互運用性の強化に加え、`WITH_RUST`ビルドフラグで有効化されるRustコードがGitリポジトリに初めて統合された。新コマンド`git last-modified`・`git repo`の追加など、Git 3.0へ向けた大規模な準備が進められている。
