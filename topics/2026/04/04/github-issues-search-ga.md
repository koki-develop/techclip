# GitHub Issuesの改良型意味ベース検索が正式一般提供、REST/GraphQL APIからも利用可能に
Tags: OSS

- Improved search for GitHub Issues is now generally available (2026-04-02)
  https://github.blog/changelog/2026-04-02-improved-search-for-github-issues-is-now-generally-available/

GitHubのIssue検索が正式一般提供となり、タイトルと本文をインデックス化した意味ベースの自然言語検索が利用可能になった。REST・GraphQL APIからも利用でき、キーワードに依存しない関連イシューの発見が可能になった。
