Anthropicのアシスタント「Claude Code」のソースコード512,000行が`.npmignore`の設定ミスによってnpmに誤って公開され、研究者のSigrid JinがそれをPythonとRustで再実装したオープンソースプロジェクト「Claw Code」を公開した。公開後数日でGitHubスター数が72,000を超え、GitHub史上最も速く成長するリポジトリの一つとなった。
