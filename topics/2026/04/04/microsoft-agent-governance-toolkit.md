# MicrosoftがAIエージェントのランタイムセキュリティOSS「Agent Governance Toolkit」を公開
Tags: AI, OSS

- Introducing the Agent Governance Toolkit: Open-Source Runtime Security for AI Agents (2026-04-02)
  https://opensource.microsoft.com/blog/2026/04/02/introducing-the-agent-governance-toolkit-open-source-runtime-security-for-ai-agents/

MicrosoftがAIエージェントのランタイムセキュリティを提供するオープンソースプロジェクト「Agent Governance Toolkit」を公開した。サブミリ秒のポリシーエンジン、暗号化されたエージェントID、ランタイムアイソレーションなどを備え、OWASP Agentic Top 10の全リスクに対応し、LangChain・AutoGen・Azure等と統合可能なPython/Rust/TypeScript/Go/.NETに対応した7パッケージ構成のシステムである。
