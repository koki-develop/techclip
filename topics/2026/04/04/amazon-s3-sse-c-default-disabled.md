# AWS、Amazon S3のカスタマー提供キー暗号化（SSE-C）を2026年4月6日より新規バケットでデフォルト無効化へ
Tags: Cloud

- Advanced notice: Amazon S3 to disable the use of SSE-C encryption by default in April 2026 (2026-04-01)
  https://aws.amazon.com/blogs/storage/advanced-notice-amazon-s3-to-disable-the-use-of-sse-c-encryption-by-default-for-all-new-buckets-and-select-existing-buckets-in-april-2026/

AWSは2026年4月6日より、Amazon S3の新規バケットに対してカスタマー提供キーによるサーバーサイド暗号化（SSE-C）をデフォルトで無効化すると発表した。一部の既存バケットも対象となるセキュリティ強化策であり、SSE-Cを利用しているユーザーは設定の見直しが必要となる。
