# Swiftが3月エコシステムまとめを公開、Swift 6.3正式リリースとSwift 6.3.1のLinux・Windows向け開発開始を報告
Tags: Programming Languages

- What's new in Swift: March 2026 Edition (2026-03-31)
  https://www.swift.org/blog/whats-new-in-swift-march-2026/
- Development open for Swift 6.3.1 for Linux and Windows (2026-03-31)
  https://forums.swift.org/t/development-open-for-swift-6-3-1-for-linux-and-windows/85715

Swift.orgが2026年3月の総括記事を公開し、Swift 6.3のリリースやSwift BuildがSwiftメインブランチのデフォルトビルドシステムとなったことを報告した。また、Swift 6.3.1のLinux・Windows向けパッチリリースの開発が開始され、マージウィンドウは4月10日まで、リリースは4月末を予定している。
