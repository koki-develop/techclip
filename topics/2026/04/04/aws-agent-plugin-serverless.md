# AWSがAIコーディングアシスタントからサーバーレスアプリを管理できる「Agent Plugin for AWS Serverless」を発表

Tags: Cloud, AI

- AWS Weekly Roundup: AWS AI/ML Scholars program, Agent Plugin for AWS Serverless, and more (March 30, 2026) (2026-03-30)
  https://aws.amazon.com/blogs/aws/aws-weekly-roundup-aws-ai-ml-scholars-program-agent-plugin-for-aws-serverless-and-more-march-30-2026/

AWSがClaude CodeやCursorなどのAIコーディングアシスタントからサーバーレスアプリのビルド・デプロイ・管理を可能にする「Agent Plugin for AWS Serverless」を発表した。あわせてAWS Lambdaのファイルディスクリプタ上限が1,024から4,096へ4倍に拡大され、大規模サーバーレスワークロードの信頼性も向上した。
