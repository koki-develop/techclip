# AIツールが開発者の言語選択を変容、TypeScriptがGitHubのコントリビューター数で首位に

Tags: AI, Programming Languages

- GitHub Data Shows AI Tools Creating "Convenience Loops" That Reshape Developer Language Choices (2026-03-25)
  https://www.infoq.com/news/2026/03/ai-reshapes-language-choice/

GitHubのデータ分析により、AIコーディングアシスタントが「利便性ループ」を形成し、TypeScriptのような静的型付け言語の採用を加速させていることが明らかになった。TypeScriptはGitHub上で初めてコントリビューター数No.1の言語となり、AIとの相性が良い言語設計の重要性が高まっている。
