# Solo.ioがkagentをCNCFに寄贈し、MCPプロトコル対応の新ツール「MCP Gateway」を発表
Tags: Cloud, AI

- Solo.io Donates Kagent to CNCF, Introduces MCP Gateway (2026-03-25)
  https://thenewstack.io/solo-io-donates-kagent-to-cncf-introduces-mcp-gateway/

Solo.ioはKubernetes上でAIエージェントを実行するフレームワーク「kagent」をCNCFサンドボックスプロジェクトとして寄贈し、AnthropicのModel Context Protocol（MCP）に対応した新ツール「MCP Gateway」を発表した。MCP GatewayはKubeCon Europe 2026で披露され、複数のMCPツールサーバーを単一の安全なエンドポイントに集約する機能を持つ。
