# CiscoのソースコードがTrivyサプライチェーン攻撃経由で窃取される

Tags: Security

- Cisco source code stolen in Trivy-linked dev environment breach (2026-03-31)
  https://www.bleepingcomputer.com/news/security/cisco-source-code-stolen-in-trivy-linked-dev-environment-breach/

Trivyサプライチェーン攻撃で盗まれた認証情報を利用した脅威アクターがCiscoの内部開発環境に侵入し、同社および顧客のソースコードを窃取した。この攻撃はGitHub・PyPI・npm・Dockerを標的とする一連のサプライチェーン攻撃を行うTeamPCPグループによるものとされている。
