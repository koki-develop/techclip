# Google Cloudが最新クラウド脅威動向レポート「Threat Horizons」を公開、第三者ソフト脆弱性が侵害の44.5%を占める
Tags: Cloud, Security

- Cloud CISO Perspectives: New Threat Horizons report highlights current cloud threats (2026-03-25)
  https://cloud.google.com/blog/products/identity-security/cloud-ciso-perspectives-new-threat-horizons-report-highlights-current-cloud-threats

Google Cloudが2026年初頭のクラウド脅威動向をまとめたThreat Horizonsレポートを発表した。サードパーティソフトウェアの脆弱性が侵害の44.5%を占め脆弱な認証情報（27.2%）を超えたこと、北朝鮮系脅威アクターがKubernetesの特権コンテナを悪用して暗号資産を窃取したキャンペーンが報告されている。
