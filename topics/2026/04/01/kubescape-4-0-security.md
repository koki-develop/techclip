# Kubescape 4.0がリリース、KubernetesのランタイムセキュリティとAIエージェントスキャンが一般提供へ
Tags: Cloud

- Kubescape 4.0 Brings Runtime Security and AI Agent Scanning to Kubernetes (2026-03-25)
  https://www.infoq.com/news/2026/03/kubescape-40/

オープンソースのKubernetesセキュリティプラットフォーム「Kubescape 4.0」がKubeCon + CloudNativeCon Europe 2026で発表・リリースされ、ランタイム脅威検知機能とAIエージェント向けセキュリティスキャンが一般提供（GA）となった。CVEノイズを95%以上削減できるとされており、Kubernetesクラスターのセキュリティ管理を大幅に効率化する。
