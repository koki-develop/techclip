# GitHub Copilot CLIがBYOK・ローカルモデルに対応、エアギャップ環境でも利用可能に
Tags: OSS

- Copilot CLI now supports BYOK and local models (2026-04-07)
  https://github.blog/changelog/2026-04-07-copilot-cli-now-supports-byok-and-local-models/

GitHub Copilot CLIが独自のモデルプロバイダーやローカルモデルの利用に対応した。OpenAI、Anthropic、Azure OpenAI互換のエンドポイントやOllama・vLLMなどのローカルモデルを設定でき、COPILOT_OFFLINE=trueでエアギャップ環境でも動作可能になった。
