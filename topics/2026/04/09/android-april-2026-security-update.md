# Android 2026年4月セキュリティ更新でゼロインタラクション脆弱性を修正
Tags: Security

- 致命的な脆弱性に対処したAndroid OSの2026年4月セキュリティ更新が発表 (2026-04-07)
  https://forest.watch.impress.co.jp/docs/news/2099692.html

Googleが2026年4月のAndroidセキュリティ情報を公開し、Frameworkコンポーネントに「Critical」評価のゼロインタラクション脆弱性（CVE-2026-0049）を修正した。ユーザー操作なしでローカルDoS攻撃が可能な脆弱性で、Android 14/15/16が影響を受ける。
