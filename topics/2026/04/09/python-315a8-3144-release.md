# Python 3.15.0a8（最終アルファ）・3.14.4・3.13.13が同時リリース
Tags: Programming Languages

- Python 3.15.0a8, 3.14.4 and 3.13.13 are out! (2026-04-07)
  https://blog.python.org/2026/04/python-3150a8-3144-31313/

Python 3.15.0a8（最終アルファ版）、Python 3.14.4（337件のバグ修正）、Python 3.13.13が同時リリースされた。3.15.0a8では遅延インポート（Explicit lazy imports）、新組み込み型frozendict、JITコンパイラ改善などの新機能が含まれる。
