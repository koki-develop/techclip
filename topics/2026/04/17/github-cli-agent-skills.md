# GitHub CLIにエージェントスキル管理機能「gh skill」を追加
Tags: OSS

- Manage agent skills with GitHub CLI (2026-04-16)
  https://github.blog/changelog/2026-04-16-manage-agent-skills-with-github-cli/

GitHub CLIに新コマンド「gh skill」が追加され、AIエージェント向けのスキルをリポジトリから検索・インストール・管理・公開できるようになった。GitHub Copilot、Claude Code、Cursor、Codex、Gemini CLIなど複数のエージェントホストに対応する。
