# Grafana v13.0リリース、ダイナミックダッシュボードとGit Syncが正式GA
Tags: OSS

- What's new in Grafana v13.0 (2026-04-14)
  https://grafana.com/docs/grafana/latest/whatsnew/whats-new-in-v13-0/

Grafana 13.0がリリースされ、次世代ダッシュボード機能「Dynamic Dashboards」とGitOps連携を実現する「Git Sync」が正式GA（一般提供）となった。新しいゲージビジュアライゼーションや可視化サジェスト機能の改善も含まれ、可観測性ワークフローの強化が図られている。
