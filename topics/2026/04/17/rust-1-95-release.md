# Rust 1.95.0リリース、cfg_select!マクロやif-letガード安定化
Tags: Programming Languages, OSS

- Announcing Rust 1.95.0 (2026-04-16)
  https://blog.rust-lang.org/2026/04/16/Rust-1.95.0/
- Rust 1.95 Released With Several Improvements (2026-04-16)
  https://www.phoronix.com/news/Rust-1.95-Released

Rust 1.95.0がリリースされ、コンパイル時にcfg条件を分岐できる新マクロ`cfg_select!`や、match式内でif-letガードが利用可能になるなどの新機能が安定化された。カスタムJSONターゲット仕様の安定版サポートが削除された点も注目される。
