# Microsoft、ノルウェーStargateデータセンターを引き継ぎNvidia Vera Rubin GPU 3万基を展開
Tags: Cloud, AI

- Microsoft Takes Over OpenAI's Norway Stargate Data Center, Deploying 30,000 Nvidia Vera Rubin GPUs (2026-04-15)
  https://www.cnbc.com/2026/04/15/openai-stargate-norway-project-microsoft.html

MicrosoftがOpenAIの手を離れたノルウェーNarvikのStargateデータセンターキャンパスを引き継ぎ、NscaleとともにNvidia Vera Rubin GPU 3万基以上を展開すると発表した。同拠点では230MWの電力容量が確保されており、Microsoftの大規模AI推論インフラ整備が加速している。
