# GitHub Copilot CLIのリモートセッション操作機能がパブリックプレビュー
Tags: OSS

- Remote control CLI sessions on web and mobile in public preview (2026-04-13)
  https://github.blog/changelog/2026-04-13-remote-control-cli-sessions-on-web-and-mobile-in-public-preview/

GitHub Copilot CLIのセッションをWebブラウザやモバイルアプリからリアルタイムで監視・操作できる「リモートCLIセッション」機能がパブリックプレビューとして公開された。`copilot --remote`で起動し、別デバイスからフォローアップ指示を送ることが可能。
