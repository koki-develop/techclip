# Slack CLI v4.0.0リリース、AIエージェント開発機能を搭載
Tags: OSS

- Slack CLI v4.0.0 and v4.0.1 release (2026-04-10)
  https://docs.slack.dev/changelog/2026/04/10/slack-cli/

Slack CLIのメジャーバージョン4.0.0がリリースされ、`slack create agent`コマンドによるAIエージェントのスキャフォールディングやSlack MCPサーバーの組み込みが追加された。Claude Agent SDK・OpenAI Agents SDK・Pydantic AIの各フレームワーク向けスターターテンプレートも提供される。
