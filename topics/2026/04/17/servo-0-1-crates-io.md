# Servo 0.1.0がcrates.ioに公開、Rustで組み込み可能なブラウザエンジン
Tags: OSS, Programming Languages

- Servo 0.1.0がcrates.ioに公開 (2026-04-13)
  https://servo.org/blog/2026/04/13/servo-0.1.0-release/

ServoプロジェクトがRust製ブラウザエンジンのv0.1.0をcrates.ioに公開し、`cargo add servo`で組み込み利用が可能になった。LTS版の提供も開始され、半年ごとのメジャーアップグレードとセキュリティアップデートの長期サポートが利用できる。
