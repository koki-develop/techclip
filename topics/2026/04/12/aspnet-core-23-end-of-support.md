# MicrosoftがASP.NET Core 2.3のサポート終了を発表、2027年4月にEOL
Tags: Programming Languages

- ASP.NET Core 2.3 end of support announcement (2026-04-07)
  https://devblogs.microsoft.com/dotnet/aspnet-core-2-3-end-of-support/
- Microsoft calls time on ASP.NET Core 2.3 on .NET Framework (2026-04-08)
  https://www.theregister.com/2026/04/08/aspnet_net_framework/

MicrosoftがASP.NET Core 2.3（.NET Framework上）のサポート終了を2027年4月7日と発表した。Entity Framework 2.3パッケージも同日にサポート終了となり、.NET 10への移行とGitHub Copilot modernizationツールの活用が推奨されている。
