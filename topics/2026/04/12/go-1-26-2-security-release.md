# Go 1.26.2/1.25.9セキュリティリリース、10件のCVEを修正
Tags: Programming Languages, Security

- [security] Go 1.26.2 and Go 1.25.9 are released (2026-04-07)
  https://groups.google.com/g/golang-announce/c/0uYbvbPZRWU

Go 1.26.2およびGo 1.25.9がセキュリティ修正を含む形でリリースされた。crypto/tls、crypto/x509、html/template、osパッケージなどに影響する10件のCVEが修正されており、インターネット公開サービスでは優先度の高いアップデートとなる。
