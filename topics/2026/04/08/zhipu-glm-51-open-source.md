# 中国Zhipu AIがMITライセンスのオープンソースLLM「GLM-5.1」を公開、SWE-Bench ProでClaude Opus・GPT-5を上回る
Tags: AI

  https://venturebeat.com/technology/ai-joins-the-8-hour-work-day-as-glm-ships-5-1-open-source-llm-beating-opus-4
- Zhipu AI launches open-source GLM-5.1 model for coding tasks (2026-04-07)
  https://www.testingcatalog.com/zhipu-ai-launches-open-source-glm-5-1-model-for-coding-tasks/

中国のZhipu AI（智谱AI）がMITライセンスのオープンソース大規模言語モデル「GLM-5.1」を公開した。ソフトウェアエンジニアリングベンチマーク「SWE-Bench Pro」においてClaude Opus 4.6やGPT-5.4を上回る性能を示すと主張しており、コーディング特化の能力でオープンソースLLMの新たな水準を示している。
