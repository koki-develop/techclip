# AWS が「Amazon S3 Files」を発表、S3バケットをNFSファイルシステムとしてマウント可能に
Tags: Cloud

- Launching S3 Files, making S3 buckets accessible as file systems (2026-04-07)
  https://aws.amazon.com/blogs/aws/launching-s3-files-making-s3-buckets-accessible-as-file-systems/
- AWS adds file access to S3, taking on NetApp and Qumulo (2026-04-07)
  https://www.blocksandfiles.com/public-cloud/2026/04/07/aws-adds-file-access-to-s3-taking-on-netapp-and-qumulo/5214675

AWSはAmazon S3 Filesを発表し、S3バケットをNFS v4.1/4.2プロトコル経由でネイティブなファイルシステムとしてマウントできる新機能を提供開始した。EC2・ECS・EKS・Lambdaなどあらゆるコンピューティングリソースからデータ移行なしに利用可能で、AIエージェントのマルチエージェントパイプラインにも対応する。業界アナリストはこの動きが、NetAppやQumuloなどエンタープライズファイルストレージ市場への本格参入を意味すると指摘している。
