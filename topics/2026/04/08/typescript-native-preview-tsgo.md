# MicrosoftがGoで書き直したTypeScriptコンパイラ「tsgo」のネイティブプレビューをnpmで公開
Tags: Programming Languages

- Announcing TypeScript Native Previews (2026-04-01)
  https://devblogs.microsoft.com/typescript/announcing-typescript-native-previews/

MicrosoftがTypeScriptコンパイラをGoで書き直した「tsgo」のネイティブプレビューをnpmパッケージ（@typescript/native-preview）として公開した。VS Codeのコードベースで従来比10倍超のコンパイル速度向上を実証しており、TypeScript 7に向けた重要なマイルストーンとなる。言語仕様はTypeScriptと完全互換を維持しつつ、ネイティブバイナリによる大幅な性能改善を実現している。
