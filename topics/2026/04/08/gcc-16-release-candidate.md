# GCC 16 のリリース候補が4月中旬を目標に準備中、C++26静的リフレクション対応など新機能を搭載予定
Tags: Programming Languages

- GCC 16 Compiler Aiming For Mid-April Release Candidate But "Slow" Progress On Fixes (2026-04-02)
  https://www.phoronix.com/news/GCC-16-RC-Mid-April

GCC 16のリリース候補（RC）が4月中旬を目標として準備中であることが報告された。C++26の静的リフレクション対応やC++20のデフォルト標準化、AMD Zen 6 / Intel AVX10.2サポートなどの新機能が含まれる予定だが、回帰テストの修正が遅れており、スケジュールに影響が出る可能性があると開発チームが示している。
