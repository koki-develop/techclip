# WordPress 7.0がリアルタイムコラボ機能の課題で延期、esbuildベースの新ビルドツールを導入
Tags: Programming Languages, OSS

- What's new for developers? (April 2026) (2026-04-10)
  https://developer.wordpress.org/news/2026/04/whats-new-for-developers-april-2026/

WordPress開発者向け月次まとめが公開され、esbuildベースの新ビルドツール「@wordpress/build」の導入が発表された。WordPress 7.0はリアルタイムコラボレーション機能のアーキテクチャ問題によりリリースが延期され、PHP 7.2/7.3のサポート終了も告知されている。
