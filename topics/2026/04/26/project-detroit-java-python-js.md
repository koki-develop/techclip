# OracleのProject DetroitがJVM内にPythonとJavaScriptを直接統合、OpenJDKプロジェクトとして開発継続
Tags: Programming Languages

- Project Detroit: bridging Java, Python, JavaScript, moves forward (2026-04-19)
  https://www.infoworld.com/article/4145953/project-detroit-bridging-java-python-javascript-moves-forward.html

OracleのProject DetroitはJVM内にV8（JavaScript）とCPython（Python）を直接組み込み、JavaからJavaScript・Pythonを高速呼び出し可能にする取り組みで、JavaOneでの発表後もOpenJDKプロジェクトとして正式化が進んでいる。クロスランゲージのインタロップを低レイテンシで実現することで、多言語混在環境の開発体験を大幅に改善することを目指している。
