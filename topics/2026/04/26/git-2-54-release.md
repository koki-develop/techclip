# Git 2.54リリース、設定ファイルでフックを定義できる「Configuration-Based Hooks」を追加
Tags: OSS, Programming Languages

- Highlights from Git 2.54 (2026-04-20)
  https://github.blog/open-source/git/highlights-from-git-2-54/

Git 2.54が137名以上のコントリビューターによってリリースされた。設定ファイル内でフックを定義できる「Configuration-Based Hooks」と、インタラクティブリベースより簡易にコミット履歴を書き換えられる実験的コマンド `git history` が主な新機能として追加されている。
