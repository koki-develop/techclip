# MetaがRust製Python型チェッカー「Pyrefly」をv0.62.0でリリース、毎秒185万行超を処理
Tags: Programming Languages, OSS

- Pyrefly: A fast, open-source type checker for Python (2026-04-20)
  https://pyrefly.org/

MetaがRustで開発した高速Python型チェッカー「Pyrefly」がv0.62.0をリリースした。毎秒185万行以上の型チェックが可能でInstagramの2,000万行コードベースを約30秒で処理でき、Python型準拠テストスイートの90%をパスしてmypy・Pyrightに並ぶ実用レベルに達している。
