# DropboxがGitHubと協力してモノレポを87GBから20GBに削減、クローン時間を1時間以上から15分以内に短縮
Tags: OSS

- Dropbox Collaborates with GitHub to Reduce Monorepo Size from 87GB to 20GB (2026-04-22)
  https://www.infoq.com/news/2026/04/dropbox-reduces-git-optimization/

DropboxがGitHubと協力し、Gitのデルタ圧縮の最適化によってバックエンドモノレポのサイズを87GBから20GBへ約77%削減することに成功した。クローン時間が1時間以上から15分以内に短縮され、CI/CDのパフォーマンスと開発者の生産性が大幅に向上した。
