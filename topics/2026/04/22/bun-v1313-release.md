# Bun v1.3.13リリース、テスト並列実行やメモリ使用量17倍削減などの大型改善
Tags: Programming Languages

- Bun v1.3.13リリース、テスト並列実行やメモリ使用量17倍削減など大型改善 (2026-04-20)
  https://bun.com/blog/bun-v1.3.13

JavaScriptランタイムBunのv1.3.13がリリースされ、テストランナーに--parallelや--shardによる並列実行機能が追加されたほか、bun installのメモリ使用量が17倍削減された。zlib-ngの統合でgzip圧縮が最大5.5倍高速化され、JavaScriptCoreエンジンの大規模アップグレードも含まれている。
