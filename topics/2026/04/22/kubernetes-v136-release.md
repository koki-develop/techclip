Kubernetes v1.36が2026年4月22日にリリースされた。80件のエンハンスメントを含み、HPAのゼロスケール対応、Linux User Namespacesの安定化、Dynamic Resource Allocationの大幅改善、OCI VolumeSourceの安定化などが含まれる。また、Ingress-NGINXの廃止に伴いGateway APIへの移行が推奨されている。
