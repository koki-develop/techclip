# WSO2がAIエージェント対応オープンソースAPIプラットフォームをGA提供開始
Tags: OSS, AI

- WSO2 Launches Open Source API Platform To Make APIs Agent-Ready Instantly (2026-04-20)
  https://www.opensourceforu.com/2026/04/wso2-launches-open-source-api-platform-to-make-apis-agent-ready-instantly/

WSO2がApache 2.0ライセンスのオープンソースAPIプラットフォームの一般提供を開始した。AIエージェントがMCP（Model Context Protocol）経由でAPIにアクセスできるAI Gatewayを中核に据え、従来のAPIとAIアセットを単一のコントロールプレーンで統合管理できる。
