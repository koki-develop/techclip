# AWS Interconnect - multicloudが一般提供開始、Google Cloudとの専用接続が本番利用可能に
Tags: Cloud

- AWS announces general availability of AWS Interconnect - multicloud (2026-04-15)
  https://aws.amazon.com/about-aws/whats-new/2026/04/aws-announces-ga-AWS-interconnect-multicloud/
- AWS Interconnect is now generally available, with a new option to simplify last-mile connectivity (2026-04-15)
  https://aws.amazon.com/blogs/aws/aws-interconnect-is-now-generally-available-with-a-new-option-to-simplify-last-mile-connectivity/

AWSがマルチクラウド接続サービス「AWS Interconnect - multicloud」の一般提供（GA）を開始した。Google Cloudを最初のパートナーとし、米東・西部など5リージョンペアで利用可能で、専用帯域による低遅延のプライベートネットワーク接続を提供する。
