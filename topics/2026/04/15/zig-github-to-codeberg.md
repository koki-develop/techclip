# ZigプログラミングがGitHubからCodebergへ公式リポジトリを移行、AI利用拒否の方針を堅持
Tags: Programming Languages, OSS

- Migrating from GitHub to Codeberg ⚡ Zig Programming Language (2026-04-14)
  https://ziglang.org/news/migrating-from-github-to-codeberg/

Zigプログラミング言語プロジェクトがGitHubからCodebergへの公式リポジトリ移行を発表した。GitHubのCopilotなどAI関連への過度な注力とGitHub Actions品質の低下が主な理由で、ZigはLLM/AIを一切使用しないポリシーを掲げている。
