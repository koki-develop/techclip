# Next.js v16.2.3リリース、React Server ComponentsのDoS脆弱性を修正
Tags: Programming Languages, Security

- Next.js v16.2.3 (2026-04-08)
  https://github.com/vercel/next.js/releases/tag/v16.2.3

Next.js v16.2.3がリリースされ、React Server ComponentsにおけるDoS脆弱性（CVE-2026-23869、CVSS 7.5）への対応を含む複数のセキュリティ修正がバックポートされた。ISR再検証のエラー報告修正やstyled-jsxの競合状態修正なども含まれている。
