# AWS DevOps AgentがGA、会話型AIによるインフラクエリとインシデント対応を提供開始
Tags: Cloud, AI

- AWS DevOps Agent is now generally available (2026-03-31)
  https://aws.amazon.com/about-aws/whats-new/2026/03/aws-devops-agent-generally-available/

AWSがDevOps Agentを一般提供（GA）とし、会話型AIによるインフラへの問い合わせやインシデント調査・対応を自動化する機能を提供開始した。マルチクラウド環境への対応も拡張されており、AWSリソースに関する自然言語クエリやアラート分析を通じてDevOpsチームの運用効率向上を支援する。
