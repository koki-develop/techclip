# JetBrainsのKotlin/Java向けビルドツールAmper 0.10がリリース、JDK自動プロビジョニングとMaven変換ツールを追加
Tags: OSS, Programming Languages

- Amper 0.10 – JDK Provisioning, a Maven Converter, Custom Compiler Plugins, and More (2026-03-31)
  https://blog.jetbrains.com/amper/2026/03/amper-0-10/

JetBrainsのKotlin/Java向けビルドツール「Amper」のバージョン0.10がリリースされ、JDKの自動プロビジョニング機能、MavenプロジェクトからAmperへの変換ツール、カスタムKotlinコンパイラプラグインのサポートが追加された。既存のMavenプロジェクトから容易に移行できる点が強調されており、Gradleの代替を目指すビルドシステムとしての成熟度が高まっている。
