# OracleのマイクロサービスフレームワークHelidon 4.4.0がリリース、LangChain4jによるAIエージェントサポートを追加
Tags: Programming Languages, AI

- Helidon 4.4.0 Introduces Alignment with OpenJDK Cadence and Support via Java Verified Portfolio (2026-04-02)
  https://www.infoq.com/news/2026/04/helidon-4-4-released/

OracleのJava向けマイクロサービスフレームワークHelidon 4.4.0がリリースされ、Java Verified Portfolio（JVP）への認定対応、新しいJSON処理ライブラリ「Helidon JSON」の導入、LangChain4jを活用したエージェント型AIサポートが追加された。OpenJDKリリースサイクルとの連携も強化され、Java 25へのスムーズな対応が図られている。
