# TigerFSがPostgreSQLデータベースをファイルシステムとしてマウント可能にするOSSツールを公開
Tags: OSS, Cloud

- TigerFS Mounts PostgreSQL Databases as a Filesystem for Developers and AI Agents (2026-04-04)
  https://www.infoq.com/news/2026/04/tigerfs-postgresql-filesystem/

TigerFSはPostgreSQLデータベースをディレクトリとしてマウントし、`ls`や`cat`などの標準Unixコマンドでデータベースのテーブルやレコードを操作できる実験的なオープンソースファイルシステムをMITライセンスで公開した。AIエージェントの状態共有用途にも活用できる設計となっており、開発者がデータベースに対して直感的にファイル操作感覚でアクセスできる新しいアプローチとして注目されている。
