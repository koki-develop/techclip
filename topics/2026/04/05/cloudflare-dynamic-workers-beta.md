# CloudflareがDynamic Workers Open Betaを公開、AIエージェントのコードをV8アイソレートで安全に実行
Tags: Cloud

- Cloudflare Launches Dynamic Workers Open Beta: Isolate-Based Sandboxing for AI Agent Code Execution (2026-04-01)
  https://www.infoq.com/news/2026/04/cloudflare-dynamic-workers-beta/

CloudflareがDynamic Worker Loaderをオープンベータとして公開し、AIエージェントが生成したコードをV8アイソレートベースのサンドボックスで安全に実行できる機能を提供開始した。コンテナと比べて起動が約100倍高速でメモリ効率も10〜100倍優れるとされており、AIコーディングエージェントのコード実行基盤として注目される。
