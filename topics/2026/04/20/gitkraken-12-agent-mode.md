# GitKraken Desktop 12.0がAIエージェント並列管理の「Agent Mode」を搭載
Tags: OSS

- GitKraken Desktop 12.0 Introduces Agent Mode: Gives Developers Ultimate Control & Visualization While Scaling Parallel Agent Workflows (2026-04-16)
  https://www.prnewswire.com/news-releases/gitkraken-desktop-12-0-introduces-agent-mode-gives-developers-ultimate-control--visualization-while-scaling-parallel-agent-workflows-302745055.html

GitKraken Desktop 12.0がリリースされ、AIコーディングエージェントの並列セッションを一元管理できる「Agent Mode」を搭載した。Claude CodeやCopilotなどのエージェントの状態をリアルタイムに可視化し、ワークツリー単位でのセッション管理が可能になった。
