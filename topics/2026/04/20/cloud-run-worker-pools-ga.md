# Google Cloud Run Worker Poolsが一般提供開始、サーバーレスで非リクエスト型ワークロードに対応
Tags: Cloud

- Google Cloud Run Worker Pools reaches General Availability (2026-04-14)
  https://cloud.google.com/run/docs/release-notes

Google Cloud RunのWorker Pools機能が一般提供（GA）に到達した。Pub/SubやKafkaからのメッセージプル処理や分散LLM学習といった非リクエスト型ワークロードをサーバーレスで実行可能になり、従来のServicesやJobsと比べて約40%のコスト削減が見込める。
