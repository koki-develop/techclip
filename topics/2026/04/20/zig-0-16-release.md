# Zig 0.16.0がリリース、新I/Oインターフェースや「Juicy Main」依存性注入を導入
Tags: Programming Languages

- 0.16.0 Released -- Zig Programming Language (2026-04-14)
  https://ziglang.org/news/0.16.0-released/
- Zig 0.16.0 released [LWN.net] (2026-04-14)
  https://lwn.net/Articles/1067634/
- Zig 0.16.0 release notes: "Juicy Main" (2026-04-15)
  https://simonwillison.net/2026/Apr/15/juicy-main/

Zig 0.16.0が8ヶ月の開発期間を経てリリースされた。I/Oをインターフェースとして扱う新しいstd.Io、main()関数への依存性注入機能「Juicy Main」、インクリメンタルコンパイルの大幅改善、パッケージ管理のローカルzig-pkgディレクトリ移行など多数の重要な変更が含まれている。
