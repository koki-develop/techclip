# FossIDがAIエージェント対応のソフトウェア構成分析「Agentic SCA」を発表
Tags: OSS, Security

- FossID launches Agentic SCA to bring real-time compliance to AI-driven code development (2026-04-14)
  https://siliconangle.com/2026/04/14/fossid-launches-agentic-sca-bring-real-time-compliance-ai-driven-code-development/

FossIDがMCP（Model Context Protocol）サーバーを通じてAIエージェントにリアルタイムのオープンソースライセンス・脆弱性・著作権チェック機能を提供する「Agentic SCA」を発表した。コード生成時点でのコンプライアンス確保を実現し、自動車・半導体・通信業界でパイロット中。
