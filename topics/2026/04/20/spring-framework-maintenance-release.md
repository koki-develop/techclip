# Spring Framework 6.2.18および7.0.7がリリース
Tags: Programming Languages

- Spring Framework 6.2.18 and 7.0.7 Available Now (2026-04-17)
  https://spring.io/blog/2026/04/17/spring-framework-6-2-18-and-7-0-7-available-now/

Spring Framework 6.2.18（27件の修正）および7.0.7（52件の修正）がリリースされた。バグ修正とドキュメント改善を中心としたメンテナンスリリースで、6.2.18は次週のSpring Boot 3.5.14に同梱される予定。
