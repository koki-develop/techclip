# AWS App Runnerが4月30日で新規受付停止、複数サービスの段階的廃止を発表
Tags: Cloud

- AWS App Runnerが4月30日で新規受付停止、メンテナンスモードへ。AWSが複数のサービス終了計画発表 (2026-04-14)
  https://www.publickey1.jp/blog/26/aws_app_runner430amazon_rds_custom_for_oracle1aws.html

AWSがAWS App Runnerを4月30日より新規受付停止・メンテナンスモードへ移行すると発表した。Amazon RDS Custom for Oracleも2027年3月31日に終了予定で、Amazon WorkMailなど複数のサービスも段階的廃止が予定されている。利用中のユーザーは代替サービスへの移行が必要となる。
