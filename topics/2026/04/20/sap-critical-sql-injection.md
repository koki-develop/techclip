# SAPの4月パッチでCVSS 9.9の重大SQLインジェクション脆弱性を修正
Tags: Security

- SAP Patchday: One critical SQL injection vulnerability – and 18 others (2026-04-14)
  https://www.heise.de/en/news/SAP-Patchday-One-critical-SQL-injection-vulnerability-and-18-others-11256718.html

SAPが4月のセキュリティパッチでCVSS 9.9の重大なSQLインジェクション脆弱性CVE-2026-27681を修正した。SAP BPC/BWの低権限ユーザーが任意のSQL文を実行し、財務データの窃取やシステム全体の侵害が可能となる深刻な欠陥である。
