# Microsoft Foundry Localが正式リリース、ローカルAI環境をインストーラで配布可能に
Tags: AI, OSS

- マイクロソフト、ローカルAI環境をインストーラで配布できる「Foundry Local」正式リリース。MacやLinuxにも対応 (2026-04-13)
  https://www.publickey1.jp/blog/26/aifoundry_localmaclinux.html

マイクロソフトがクラウド不要でアプリケーションにバンドル・配布できるローカルAI環境「Foundry Local」を正式リリースした。Windows・Mac・Linuxのクロスプラットフォームに対応し、GPT OSS・Qwen・Deepseekなど多様なモデルから選択可能で、オフライン環境でのAI活用を推進する。
