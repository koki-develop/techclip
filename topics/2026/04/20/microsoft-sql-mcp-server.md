# MicrosoftがPostgreSQL/MySQL/SQL Server対応の「SQL MCP Server」をオープンソースで公開
Tags: OSS, Cloud

- マイクロソフト、PostgreSQL/MySQL/SQL Serverなどへの同時接続に対応した「SQL MCP Server」オープンソースで公開 (2026-04-16)
  https://www.publickey1.jp/blog/26/postgresqlmysqlsql_serversql_mcp_server.html

マイクロソフトが、PostgreSQL・MySQL・SQL Serverへの同時接続に対応したオープンソースの「SQL MCP Server」を公開した。AIエージェントがModel Context Protocol経由で複数のデータベースに対して横断的に問い合わせを実行できるようになり、MCPエコシステムの拡充に貢献している。
