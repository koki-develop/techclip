# GitHub Copilot CLIがautoモデル選択機能を正式GA
Tags: OSS

- GitHub Copilot CLI now supports Copilot auto model selection (2026-04-17)
  https://github.blog/changelog/2026-04-17-github-copilot-cli-now-supports-copilot-auto-model-selection/

GitHub Copilot CLIでautoモデル選択機能が全プランで正式GAとなった。GPT-5.4、Sonnet 4.6、Haiku 4.5などのモデルを自動ルーティングし、レート制限を軽減しつつ最適なモデルを選択する。autoモデル使用時にはモデル乗数が10%割引される。
