# GoogleのMCP Toolbox for DatabasesがModel Context Protocolに正式対応
Tags: Cloud, AI

- データベース向けMCPツールボックス、Model Context Protocolをサポート (2026-04-19)
  https://cloud.google.com/blog/ja/products/ai-machine-learning/mcp-toolbox-for-databases-now-supports-model-context-protocol?hl=ja

GoogleのデータベースアクセスツールであるMCP Toolbox for DatabasesがModel Context Protocolに正式対応した。AIエージェントがデータベースに標準的なMCPインターフェースを通じてアクセスできるようになり、開発者はデータベース連携をより効率的に実装できる。
