# 「State of Trusted Open Source Report」：脆弱性が前四半期比145%増、AIによる開発加速が影響
Tags: Security, OSS

- The State of Trusted Open Source Report (April 2026) (2026-04-02)
  https://thehackernews.com/2026/04/the-state-of-trusted-open-source-report.html

2,200以上のコンテナイメージプロジェクトと33,931件の脆弱性インスタンスを分析したオープンソースセキュリティレポートが公開された。脆弱性の件数が前四半期比145%増加する一方、修正の中央値は2.0日を維持しており、AIを活用した開発加速がセキュリティリスクの拡大にも影響していることが示された。
