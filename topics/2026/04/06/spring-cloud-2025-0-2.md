# Spring Cloud 2025.0.2（Northfields）リリース、Spring Boot 3.5.13ベースのメンテナンス版
Tags: Programming Languages

- Spring Cloud 2025.0.2 (aka Northfields) Has Been Released (2026-04-02)
  https://spring.io/blog/2026/04/02/spring-cloud-2025-0-2-aka-northfields-has-been-released/

SpringチームがSpring Cloud 2025.0.2（コードネーム「Northfields」）の一般提供を発表した。Spring Boot 3.5.13をベースとしたメンテナンスリリースで、複数のバグ修正と依存関係のアップデートが含まれる。
