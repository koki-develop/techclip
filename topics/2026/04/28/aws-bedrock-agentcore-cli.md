# AWSがBedrock AgentCore CLIプレビューを公開、AIエージェントのIaC管理が可能に

Tags: Cloud, AI

- AWS Weekly Roundup: Anthropic & Meta partnership, AWS Lambda S3 Files, Amazon Bedrock AgentCore CLI, and more (April 27, 2026) (2026-04-27)
  https://aws.amazon.com/blogs/aws/aws-weekly-roundup-anthropic-meta-partnership-aws-lambda-s3-files-amazon-bedrock-agentcore-cli-and-more-april-27-2026/

AWSはBedrock AgentCore CLIのプレビューを14リージョンで無料公開し、CDKやTerraformを利用したAIエージェントのインフラコード（IaC）管理を可能にした。同週次ラウンドアップではAWS LambdaとS3 Filesの統合、AnthropicおよびMetaとの提携拡大も発表されており、AIエージェントの本番デプロイ基盤整備が加速している。
