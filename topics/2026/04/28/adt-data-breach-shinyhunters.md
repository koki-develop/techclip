# ホームセキュリティ大手ADTがShinyHuntersに侵害、550万人分の個人情報が流出

Tags: Security

- Home security giant ADT data breach affects 5.5 million people (2026-04-27)
  https://www.bleepingcomputer.com/news/security/home-security-giant-adt-data-breach-affects-55-million-people/

ホームセキュリティ大手ADTがハッカーグループShinyHuntersによるサイバー攻撃を受け、約550万人分の個人情報（氏名・電話番号・住所を含む）約11GBのデータが漏洩した。ADTは4月20日に侵害を検知し、恐喝交渉が決裂したためデータがダークウェブ上で公開された。
