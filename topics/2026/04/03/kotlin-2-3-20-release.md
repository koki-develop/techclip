# Kotlin 2.3.20リリース、Gradle 9.3.0との互換性や名前ベース分割宣言など新機能追加
Tags: Programming Languages

- Kotlin 2.3.20 Released (2026-03-31)
  https://blog.jetbrains.com/kotlin/2026/03/kotlin-2-3-20-released/

Kotlin 2.3.20がリリースされ、Gradle 9.3.0との互換性確保、Mavenプロジェクトのセットアップ簡素化、名前ベースの分割宣言、LombokサポートのAlpha昇格などが含まれる。Kotlin/NativeにおけるCおよびObjective-Cライブラリの新しい相互運用モードも追加され、マルチプラットフォーム開発がさらに強化された。
