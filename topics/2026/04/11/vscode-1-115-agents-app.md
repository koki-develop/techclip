VS Code 1.115がリリースされ、エージェントネイティブ開発に特化した「VS Code Agents」コンパニオンアプリが新たに導入された。ターミナルへの画像ペースト、エージェントによるバックグラウンドターミナル操作、ミニマップでのテストカバレッジ表示など、開発ワークフローを大幅に強化する機能が追加された。
