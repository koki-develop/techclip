# Microsoft Windows 365クラウドPCを中小企業向けに20%値下げ
Tags: Cloud

- Microsoft Cuts Windows 365 Cloud PC Prices by 20 Percent for SMBs (2026-04-10)
  https://www.theregister.com/2026/04/10/microsoft_cuts_cloudy_desktop_prices/

Microsoftが中小企業向けにWindows 365クラウドPCの価格を5月1日から20%引き下げると発表。コスト削減と引き換えに、サインアウト後1時間でハイバネーション状態に移行する仕様変更が行われる。
