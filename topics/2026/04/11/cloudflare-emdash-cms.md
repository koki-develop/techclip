# CloudflareがTypeScriptベースのオープンソースCMS「EmDash」を発表
Tags: Programming Languages, OSS

- Cloudflare Introduces EmDash: TypeScript CMS Positioned as WordPress Successor (2026-04-09)
  https://www.infoq.com/news/2026/04/cloudflare-emdash-wordpress/

CloudflareがTypeScriptとAstro 6.0ベースのオープンソースCMS「EmDash」のプレビュー版を公開した。WordPressの「精神的後継者」と位置づけ、プラグインをDynamic Workersでサンドボックス化することで、WordPressのプラグイン起因のセキュリティ問題を根本的に解決する設計となっている。
