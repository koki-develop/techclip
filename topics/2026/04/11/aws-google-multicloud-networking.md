# AWSとGoogle Cloudがマルチクラウド接続サービスのプレビューを発表
Tags: Cloud

- AWS and Google Cloud Preview Secure Multicloud Networking (2026-04-07)
  https://www.ciodive.com/news/aws-google-link-cloud-products/806705/

AWSとGoogle Cloudが共同でマルチクラウドインターコネクトサービスのプレビューを発表した。AWSマネジメントコンソールからGoogle Cloud VPCへの専用低遅延リンクを直接プロビジョニングできるようになり、Microsoftも同年中の対応を予定しており、主要クラウド間のネイティブ接続標準化が進んでいる。
