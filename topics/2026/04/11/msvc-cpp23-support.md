# MSVC Build Tools 14.51のC++23対応状況をMicrosoftが公開
Tags: Programming Languages

- C++23 Support in MSVC Build Tools 14.51 (2026-04-09)
  https://devblogs.microsoft.com/cppblog/c23-support-in-msvc-build-tools-14-51/

MicrosoftがMSVC Build Tools 14.51におけるC++23言語機能・標準ライブラリ・ISO C++ Working Group課題への対応状況を公開した。C++23の完全サポートに向け残り2機能が開発中であり、完了次第/std:c++23スイッチが正式追加される予定。
