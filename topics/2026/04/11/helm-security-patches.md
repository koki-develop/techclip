# Helm v4.1.5/v3.20.3がセキュリティ修正をリリース、パストラバーサル等を修正
Tags: OSS, Security

- Helm v4.1.5 and v3.20.3 patch releases (2026-04-08)
  https://github.com/helm/helm/releases

Kubernetesパッケージマネージャー Helmのパッチリリースが公開され、プラグインのprovenance検証バイパス、チャート名のドット文字によるパス不正、プラグインバージョンのパストラバーサル脆弱性が修正された。セキュリティ関連の修正を含むため、速やかなアップデートが推奨される。
