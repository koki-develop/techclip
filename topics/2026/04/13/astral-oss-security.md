# Astralがuv/Ruffのオープンソースセキュリティ対策手法を詳細公開
Tags: OSS, Security

- Open source security at Astral (2026-04-08)
  https://astral.sh/blog/open-source-security-at-astral

Python向け高速ツールチェーン「uv」「Ruff」の開発元であるAstralが、自社OSSプロジェクトのセキュリティ対策手法を詳細に公開した。CI/CDワークフロー、リポジトリ管理、リリースプロセス、依存関係管理など5つの領域でのサプライチェーンセキュリティ対策を解説している。
