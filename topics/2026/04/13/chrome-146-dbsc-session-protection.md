# Google Chrome 146がDevice Bound Session Credentialsを一般提供、セッションCookie窃取を防止
Tags: Security

GoogleがChrome 146でDevice Bound Session Credentials（DBSC）をWindows向けに一般提供開始した。TPMなどのハードウェアセキュリティモジュールを使ってセッションをデバイスに紐付けることで、LummaやVidarなどのインフォスティーラーによるセッションCookie窃取を無効化する。
