# Azure MCP Server 2.0安定版リリース、57サービス276ツールでエージェントクラウド自動化に対応
Tags: OSS, Cloud

- Announcing Azure MCP Server 2.0 Stable Release for Self-Hosted Agentic Cloud Automation (2026-04-10)
  https://devblogs.microsoft.com/azure-sdk/announcing-azure-mcp-server-2-0-stable-release/
- Microsoft Announces Azure MCP Server 2.0 Stable Release for Enterprise AI Automation (2026-04-10)
  https://www.c-sharpcorner.com/news/microsoft-announces-azure-mcp-server-20-stable-release-for-enterprise-ai-automation

MicrosoftがAzure MCP Server 2.0を安定版としてリリースし、57のAzureサービスにわたる276のツールを提供する。セルフホスト対応やHTTPベースのリモートホスティング強化、セキュリティガバナンス向上など、AIエージェントワークフロー向けの基盤が大幅に改善された。
