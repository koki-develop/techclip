# GitHub Issueサイドバーにリリース情報を表示、Projectsフィールドにデフォルト値設定が可能に
Tags: OSS

- Release information in issue sidebar and default values for project fields (2026-04-09)
  https://github.blog/changelog/2026-04-09-release-info-in-issue-sidebar-and-project-defaults/

GitHub Issueのサイドバーにリリースリンクが表示されるようになり、バグ修正がいつリリースされたかをIssueページから直接確認できるようになった。また、GitHub Projectsのテキスト・数値・単一選択フィールドにデフォルト値を設定可能になった。
