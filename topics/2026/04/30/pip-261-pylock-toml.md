# pip 26.1リリース、PEP 751標準のロックファイル形式pylock.tomlを試験的サポート
Tags: Programming Languages

- Announcement: pip 26.1 release (2026-04-26)
  https://discuss.python.org/t/announcement-pip-26-1-release/107108
- What's new in pip 26.1 (2026-04-28)
  https://simonwillison.net/2026/Apr/28/pip-261/

Pythonのパッケージマネージャーpipがバージョン26.1をリリースし、PEP 751で標準化されたロックファイル形式「pylock.toml」の実験的サポートを追加した。Python 3.9のサポートが廃止されたほか、依存関係クールダウン機能（--uploaded-prior-toオプション）も新たに導入されている。
