# CISAがConnectWise ScreenConnectとWindows Shellの脆弱性をKEVカタログに追加
Tags: Security

- CISA Adds Actively Exploited ConnectWise and Windows Flaws to KEV (2026-04-29)
  https://thehackernews.com/2026/04/cisa-adds-actively-exploited.html
- CISA Adds Two Known Exploited Vulnerabilities to Catalog (2026-04-28)
  https://www.cisa.gov/news-events/alerts/2026/04/28/cisa-adds-two-known-exploited-vulnerabilities-catalog
- U.S. CISA adds Microsoft Windows Shell and ConnectWise ScreenConnect flaws to its Known Exploited Vulnerabilities catalog (2026-04-29)
  https://securityaffairs.com/191442/security/u-s-cisa-adds-microsoft-windows-shell-and-connectwise-screenconnect-flaws-to-its-known-exploited-vulnerabilities-catalog.html

CISAはConnectWise ScreenConnect（CVE-2024-1708）のパストラバーサル脆弱性とMicrosoft Windows Shell（CVE-2026-32202）の保護機構回避脆弱性を既知悪用脆弱性（KEV）カタログに追加した。連邦機関には2026年5月12日までの修正適用が義務付けられており、両脆弱性はすでに実際の攻撃で積極的に悪用されている。
