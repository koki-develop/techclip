# 北朝鮮関連ハッカーがGitHubをC2インフラとして悪用、韓国組織に多段階攻撃
Tags: Security

- DPRK-Linked Hackers Use GitHub as C2 in Multi-Stage Attacks Targeting South Korea (2026-04-06)
  https://thehackernews.com/2026/04/dprk-linked-hackers-use-github-as-c2-in.html

北朝鮮に関連する脅威アクターが、GitHubのリポジトリをC2（コマンド＆コントロール）インフラとして悪用し、韓国の組織を標的とした多段階攻撃を展開していることが判明した。フィッシングメールに含まれる難読化されたWindowsショートカットファイルを起点に、PowerShellスクリプトで持続性を確立しGitHubアカウントへホスト情報を送信する手口が確認されている。
