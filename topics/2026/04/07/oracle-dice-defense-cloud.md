# OracleがDIB Isolated Cloud Environment（DICE）を発表、防衛関連企業向け隔離クラウドを提供
Tags: Cloud

- Oracle Empowers Defense Industrial Base to Innovate at Scale with New Isolated Cloud (2026-03-31)
  https://www.oracle.com/news/announcement/oracle-empowers-defense-industrial-base-to-innovate-at-scale-with-new-isolated-cloud-2026-03-31/
- Oracle launches DIB Isolated Cloud (2026-03-31)
  https://intelligencecommunitynews.com/oracle-launches-dib-isolated-cloud/

OracleがDIB Isolated Cloud Environment（DICE）を発表し、防衛関連企業向けに機密性の高い隔離されたクラウド環境の提供を開始した。Secretレベルの暫定認可を夏までに取得予定で、複数の防衛請負業者間でのセキュアな協業を実現する。
