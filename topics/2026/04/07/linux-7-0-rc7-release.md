# Linux 7.0-rc7リリース、来週の安定版7.0公開に向け最終候補版が登場

Linus TorvaldsがLinux 7.0の最終リリース候補となる7.0-rc7を公開した。AIエージェント向けドキュメント改善やWi-Fiドライバーのパフォーマンス修正が含まれており、問題がなければ翌週に安定版7.0がリリースされる見込みで、Intel Nova LakeやAMD Zen 6など最新CPUのサポートが含まれる。
