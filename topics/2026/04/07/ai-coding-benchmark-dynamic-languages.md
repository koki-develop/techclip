# 13言語対象のClaude Codeベンチマーク：動的言語が静的型付け言語より1.4〜2.6倍安く速い
Tags: Programming Languages, AI

- Dynamic Languages Faster and Cheaper in 13-Language Claude Code Benchmark (2026-04-06)
  https://www.infoq.com/news/2026/04/ai-coding-language-benchmark/

RubyコミッターのYusuke Endoh氏が13言語を対象にClaude Codeのベンチマーク（600回実行）を行い、Ruby・Python・JavaScriptなどの動的言語が静的型付け言語と比べて1.4〜2.6倍安く速いことを示した。型チェッカーの追加はさらに1.6〜3.2倍の速度低下をもたらすことも判明し、AIコーディングエージェント時代の言語選択に新たな視点を提供している。
