# Google ADK Go 1.0正式リリース、Go言語向けAIエージェント開発キットがGA到達
Tags: Programming Languages, AI

- ADK Go 1.0 Arrives! (2026-03-31)
  https://developers.googleblog.com/adk-go-10-arrives/

GoogleがGo言語向けのAIエージェント開発キット「ADK Go 1.0」の正式リリースを発表した。OpenTelemetryによる深いトレーシング、プラグインシステム、Human-in-the-Loopによる安全制御、YAML設定サポートなど、プロダクション対応のAIエージェント開発に必要な機能が揃い、Goエコシステムでのエージェント開発が本格化する。
