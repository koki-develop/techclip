# MicrosoftがMedusaランサムウェア関連グループStorm-1175によるゼロデイ悪用攻撃を報告
Tags: Security

- Microsoft links Medusa ransomware affiliate to zero-day attacks (2026-04-06)
  https://www.bleepingcomputer.com/news/security/microsoft-links-medusa-ransomware-affiliate-to-zero-day-attacks/

Microsoftは中国を拠点とするサイバー犯罪グループ「Storm-1175」が、ゼロデイおよびnデイ脆弱性を悪用してMedusaランサムウェアを展開する高速攻撃を実施していると報告した。医療・教育・金融・専門サービス分野を標的とし、初期侵入からデータ窃取・ランサムウェア展開まで24時間以内に完了するケースもあるという。
