# 不満を持つ研究者がWindowsのローカル権限昇格ゼロデイ「BlueHammer」のPoCをリーク
Tags: Security

- Disgruntled researcher leaks "BlueHammer" Windows zero-day exploit (2026-04-06)
  https://www.bleepingcomputer.com/news/security/disgruntled-researcher-leaks-bluehammer-windows-zero-day-exploit/
- BlueHammer Windows Zero-Day Exploit Leaks Online (2026-04-07)
  https://www.technadu.com/bluehammer-zero-day-exploit-leverages-windows-privilege-escalation-prompts-security-concerns/625409/

Microsoftのバグ報告対応への不満から、セキュリティ研究者がWindowsのTOCTOU競合とパス混乱を組み合わせたローカル権限昇格ゼロデイ「BlueHammer」のPoCコードを公開した。未パッチの状態で攻撃者がSYSTEM権限へ昇格できる可能性があり、Microsoftの対応が注目されている。
