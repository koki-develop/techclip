## 主な新機能と改善点

Visual Studio Codeの2026年3月アップデートとなるバージョン1.113が3月25日にリリースされた。今回のリリースでは、エージェント機能とAI連携が大きく強化されている。

注目の機能として、Model Context Protocol（MCP）サーバーのブリッジ対応がある。VS Codeで設定されたMCPサーバーが、Copilot CLIやClaudeエージェントからも利用可能になった。ユーザー定義およびワークスペース定義の`mcp.json`ファイルによるサーバー設定がサポートされている。また、サブエージェントが他のサブエージェントを呼び出せる「ネストされたサブエージェント」機能が導入され、`chat.subagents.allowInvocationsFromSubagents`設定で有効化できる。これにより、複雑なワークフローの自動化が可能になった。

さらに、Copilot CLIやClaudeエージェントのセッションを分岐して別の方向性を試せる「セッションフォーキング」機能（実験的）や、エージェントデバッグログパネルの対応拡大なども追加されている。

## チャット体験の刷新

チャットカスタマイズエディタ（プレビュー）が新たに導入された。カスタムインストラクション、プロンプトファイル、カスタムエージェント、エージェントスキルを一元管理できる統一インターフェースで、コードエディタの埋め込みや構文バリデーション機能も備えている。

推論をサポートするモデルでは、モデルピッカーに「思考の深さ（Thinking Effort）」のサブメニューが表示されるようになり、Low/Medium/Highの3段階から選択可能になった。設定はモデルごとに会話をまたいで保持される。なお、従来の`github.copilot.chat.anthropic.thinking.effort`などの設定は非推奨となり、このUIに統合された。

画像添付のプレビュー機能も強化され、ナビゲーション矢印、サムネイルストリップ、ズーム/パン機能、会話ターンごとのセクショングルーピングなどが追加されている。

## エディタ体験とテーマの更新

新しいデフォルトテーマとして「VS Code Light」と「VS Code Dark」が導入された。モダンなデザインを採用しながらも使いやすさを維持しており、新規ユーザーにはOSのテーマ設定に合わせて自動的に適用される。

ブラウザ関連の機能も改善されており、統合ブラウザでHTTPS開発時に自己署名証明書を一時的に信頼できる機能や、ブラウザタブの管理を効率化するQuick Open Browser Tabコマンド（ショートカット: Shift+Cmd+A / Ctrl+Shift+A）が追加された。

なお、Edit Modeはv1.110で正式に非推奨となっており、v1.125で完全に削除される予定。それまでは`chat.editMode.hidden`設定で再有効化が可能となっている。
